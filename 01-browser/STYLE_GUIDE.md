# Style Guide: Google Antigravity

This guide outlines the design system used by [antigravity.google](https://antigravity.google/) to enable replication of its clean, premium, and airy aesthetic.

## 1. Color Palette

The site uses a high-contrast, minimalist palette centered around Google's core identity.

| Name | HEX Code | Usage |
| :--- | :--- | :--- |
| **Primary Black** | `#000000` | Headings, primary buttons, footer sections. |
| **Primary White** | `#FFFFFF` | Main background, primary button text. |
| **Cool Gray 1** | `#F8F9FA` | Secondary backgrounds, subtle section transitions. |
| **Cool Gray 2** | `#E8EAED` | Secondary buttons (hover state), borders. |
| **Stardust Blue** | `#4285F4` | Particle effects, subtle highlights. |

> [!NOTE]
> The logo uses the standard Google Brand Palette (Blue, Red, Yellow, Green).

## 2. Typography

The typography is modern, using variable weights to create clear hierarchy.

*   **Primary Font Face:** `Google Sans Flex`
    *   **Hero Headings:** 700 (Bold), ~64px+, -0.02em letter-spacing.
    *   **Sub-headings:** 500 (Medium), ~32px-48px.
    *   **Body Text:** 400 (Regular), ~16px-18px, 1.5 line-height.
*   **Icon Font:** `Google Symbols` (Variable font for functional UI icons).

## 3. UI Components

### Buttons
*   **Style:** Pill-shaped (border-radius: `9999px`).
*   **Primary:** Black background, White text.
*   **Secondary:** Light Gray (`#F8F9FA`) background, Black text.
*   **Interaction:** Subtle scaling (1.02) or background color shift on hover.

### Cards & Sections
*   **Containers:** Large, breathable padding.
*   **Corners:** Significant rounding (approx. `32px` border-radius) on main IDE showcases and feature cards.
*   **Borders:** Most sections are borderless, using whitespace and subtle background shifts to define areas.

### Spacing
*   **Vertical Padding:** Generous `120px` to `160px` between major sections to maintain a premium feel.
*   **Layout:** Centered max-width container (approx. `1280px`).
