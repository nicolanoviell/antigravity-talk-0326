Motion & Interactivity: Implement all animations and transitions (on scroll, hover, and page load) exactly as detailed in ANIMATIONS.md.

Content Structure: The blog should include a header/navigation, a hero section, a grid of blog posts (cards), and a footer.

Technical Stack: Use clean HTML5 and modern CSS (you may use Tailwind CSS or Vanilla CSS as per the style guide's suggestions). If complex animations are required, use a lightweight library like Lucide icons or simple CSS Keyframes.

Output: Provide the complete, production-ready code (HTML and CSS) in a single block or separate files if necessary.
